The Student Result Processing System is a C-based application developed to process student academic results efficiently.
This project is a modularized version of the system developed earlier, enhanced with software quality characteristics, proper documentation, and a test plan, as required in Task-2 of the Software Engineering Lab
To modularize the application into independent, reusable components

To improve software quality characteristics such as usability and efficiency

To apply software engineering principles like low coupling and high cohesion

To document modules and test cases

To maintain the project using Git and GitHub
